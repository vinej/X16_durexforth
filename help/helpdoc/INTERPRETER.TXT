=== INTERPRETER - Interpreter and input source ===

[x] TIB ( -- addr ) - the terminal input buffer.
[x] #TIB ( -- addr ) - variable: number of characters in TIB.
[x] >IN ( -- addr ) - variable: current parse offset into the input.
[x] SOURCE ( -- addr len ) - the current input buffer and its length.
[x] SOURCE-ID ( -- 0 | -1 | fileid ) - 0 = keyboard, -1 = EVALUATE, else a file.
[x] REFILL ( -- flag ) - read the next input line; false at end.
[x] EVALUATE ( addr len -- ) - interpret a string as Forth. S" 2 3 +" EVALUATE .
    ->  5
[ ] SAVE-INPUT ( -- xn..x1 n ) - save the input position.
[ ] RESTORE-INPUT ( xn..x1 n -- flag ) - restore a saved input position.
[ ] CLOSE-SOURCE ( -- ) - close the current file input source (used by nested
    INCLUDE).
[x] "xxx" ( -- addr len ) - string literal, same semantics as S" but no space
    after the quote: "hello world" TYPE. Only kicks in for undefined tokens
    (defined words always win); an unterminated string errors like an
    unknown word. Handler cell: 'QUOTE (like 'NOTFOUND).
[x] 'QUOTE ( -- addr ) - the "xxx" string-literal hook cell; the kernel routes
    undefined tokens starting with " through it.
[x] STATE ( -- addr ) - variable: non-zero while compiling.
[x] ( ( "ccc)" -- ) - inline comment up to ). ( n -- n*2 )
[x] \ ( "line" -- ) - comment to end of line.
