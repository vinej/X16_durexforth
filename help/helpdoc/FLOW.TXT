=== FLOW - Control flow ===

These are IMMEDIATE and used inside : definitions.

[x] IF ( flag -- ) - begin a conditional; runs to ELSE/THEN when flag is true.
[x] THEN ( -- ) - end an IF.
[x] ELSE ( -- ) - the false branch of IF.
[x] BEGIN ( -- ) - start of a loop.
[x] UNTIL ( flag -- ) - loop back to BEGIN until flag is true.
[x] AGAIN ( -- ) - loop back to BEGIN forever.
[x] WHILE ( flag -- ) - mid-test; falls through to after REPEAT when false.
[x] REPEAT ( -- ) - loop back to BEGIN (with WHILE).
[x] DO ( limit start -- ) - begin a counted loop. 10 0 DO I . LOOP
[x] ?DO ( limit start -- ) - like DO but skips entirely if limit = start.
[x] LOOP ( -- ) - increment the index by 1 and test.
[x] +LOOP ( n -- ) - increment the index by n and test. 10 0 DO I . 2 +LOOP
[x] UNLOOP ( -- ) - discard loop control (before an early EXIT).
[x] LEAVE ( -- ) - exit the innermost loop immediately.
[x] CASE ( -- ) - begin a multi-way branch (IMMEDIATE). CASE 1 OF ." one" ENDOF ." other" ENDCASE
[x] OF ( x y -- | x ) - selector; runs its clause to ENDOF when x = y.
[x] ENDOF ( -- ) - end an OF clause (jump past ENDCASE).
[x] ENDCASE ( x -- ) - end a CASE (drops the selector).
[x] AHEAD ( -- ) - compile an unconditional forward branch (advanced).
[x] RECURSE ( -- ) - call the definition currently being compiled.
[x] EXIT ( -- ) - return from the current word.
[x] EXECUTE ( xt -- ) - call the word given by its xt. ' CR EXECUTE
[x] QUIT ( -- ) - empty the return stack and re-enter the interpreter (no
    message).
[x] ABORT ( -- ) - clear both stacks and return to the interpreter.
[x] ABORT" ( flag "ccc\"" -- ) - if flag is true, print the message and ABORT. x
    0= ABORT" zero!"
[x] CATCH ( i*x xt -- j*x 0 | i*x n ) - run xt; return 0 if it completes, or the
    code n passed to THROW, with the stacks restored. The recoverable
    alternative to ABORT. ' RISKY CATCH IF ." failed" THEN
[x] THROW ( k*x n -- k*x | i*x n ) - if n is non-zero, unwind to the nearest CATCH
    and return n from it; 0 THROW does nothing. An uncaught THROW performs
    ABORT.
[ ] SP@ ( -- addr ), RP@ ( -- addr ) - fetch the data / return stack pointer (used
    by CATCH).
[ ] HANDLER ( -- addr ) - variable holding the current exception frame (0 = none).
