=== CONTROL - X16 system control ===

[x] EDIT ( c-addr u -- ) - open the named file in the built-in X16 text editor
  (x16edit, ROM bank $0D; u=0 opens a new empty buffer). Edit, save (Ctrl-S),
  quit (Ctrl-Q), then INCLUDE the file to compile it. Typical loop:
    S" PROG.FS" EDIT   S" PROG.FS" INCLUDED
  EDIT saves/restores the data-stack zeropage the editor uses, resets the RAM
  bank to 0, closes the editor's files, and hides the mouse on exit; file I/O
  and console output after EDIT work normally.
[x] SETBANK ( bank -- ) - select the RAM bank visible at $A000-$BFFF.
[x] B@ ( bank off -- byte ) - read a byte from banked RAM (off = 0..8191 into
    $A000). Preserves the $00 bank register.
[x] B! ( byte bank off -- ) - store a byte into banked RAM. Preserves $00.
[x] SLEEP ( jiffies -- ) - wait jiffies 1/60-second ticks. 60 SLEEP waits ~1 s.
[x] MS ( u -- ) - wait ~u milliseconds (calibrated 8 MHz busy loop; approximate
    but >= u ms). 1000 MS ~ 1 s.
[x] REBOOT ( -- ) - soft reboot through the reset vector.
[ ] KEYMAP ( c-addr u -- ) - set the keyboard layout by name. S" en-us" KEYMAP

--- clock (battery RTC / jiffy timer) ---
[x] TICKS   ( -- ud )      the 24-bit jiffy counter (1/60 s) as an unsigned double.
[x] TIME@   ( -- h m s )   read the wall-clock time.
[x] DATE@   ( -- y mo d )  read the date (y = full 4-digit year).
[x] SETTIME ( y mo d h m s -- )  set the clock.  2025 7 8 14 30 0 SETTIME

--- I2C bus (SMC, RTC, and other I2C devices) ---
[x] I2CPEEK ( dev reg -- byte )   read a byte from I2C device dev, register reg.
[x] I2CPOKE ( dev reg val -- )    write val to I2C device dev, register reg.

--- power / debug ---
[x] MONITOR  ( -- )   enter the built-in machine-language monitor (exit with X;
                  it does NOT cleanly return - reset afterward).
[x] RESET    ( -- )   hardware reset via the System Management Controller.
[x] POWEROFF ( -- )   power the machine off via the SMC.
