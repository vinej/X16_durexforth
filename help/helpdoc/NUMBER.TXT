=== NUMBER - Number and text parsing ===

Literals: $FF hex, %101 binary, #10 decimal, 'c' char code, and a trailing
dot makes a double-cell literal in the current base (12. -> 12 0, see
DOUBLE). With FLOAT loaded, 12.12 / 1E6 are float literals.

[x] >NUMBER ( ud addr len -- ud' addr' len' ) - accumulate digits into a double
    until a non-digit.
[x] WORD ( c -- c-addr ) - parse the next c-delimited word into a counted string.
    BL WORD
[x] PARSE ( c -- addr len ) - parse the input up to the next c (no leading skip).
[x] PARSE-NAME ( -- addr len ) - parse the next space-delimited name.
[x] FIND ( c-addr -- c-addr 0 | xt 1 | xt -1 ) - look up a counted-string name; 1
    = immediate, -1 = normal.
