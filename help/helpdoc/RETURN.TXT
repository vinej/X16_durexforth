=== RETURN - Return stack and loop index ===

[x] >R ( x -- ) (R: -- x) - move an item to the return stack.
[x] R> ( -- x ) (R: x -- ) - move it back.
[x] R@ ( -- x ) - copy the top of the return stack.
[x] RDROP (R: x -- ) - drop the top of the return stack.
[x] 2>R ( x1 x2 -- ) - move a pair to the return stack.
[x] 2R> ( -- x1 x2 ) - move a pair back.
[x] 2R@ ( -- x1 x2 ) - copy a pair.
[x] I ( -- n ) - index of the innermost DO loop.
[x] J ( -- n ) - index of the next-outer DO loop.
