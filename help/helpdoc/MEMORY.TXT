=== MEMORY - Memory ===

Cells are 16-bit (2 bytes). Addresses are byte addresses.

[x] @ ( addr -- x ) - fetch a cell. X @
[x] ! ( x addr -- ) - store a cell. 42 X !
[x] C@ ( addr -- c ) - fetch a byte.
[x] C! ( c addr -- ) - store a byte.
[x] 2@ ( addr -- x1 x2 ) - fetch a double / two cells.
[x] 2! ( x1 x2 addr -- ) - store two cells.
[x] +! ( n addr -- ) - add n to the cell at addr. 1 X +!
[x] , ( x -- ) - compile a cell into the dictionary at HERE. CREATE T 1 , 2 , 3 ,
[x] C, ( c -- ) - compile a byte into the dictionary.
[x] HERE ( -- addr ) - next free dictionary address.
[x] ALLOT ( n -- ) - reserve n bytes of dictionary space. CREATE BUF 100 ALLOT
[x] UNUSED ( -- u ) - bytes of dictionary space remaining. UNUSED .
[x] PAD ( -- addr ) - address of a scratch buffer (not preserved across words).
[x] ERASE ( addr u -- ) - set u bytes to 0.
[x] FILL ( addr u c -- ) - set u bytes to character c.
[x] BLANK ( addr u -- ) - set u bytes to spaces.
[x] MOVE ( src dst u -- ) - copy u bytes, handling overlap correctly.
[x] CMOVE ( src dst u -- ) - copy u bytes low -> high.
[x] CMOVE> ( src dst u -- ) - copy u bytes high -> low.
[x] CELL+ ( addr -- addr+2 ) - advance one cell.
[x] CELLS ( n -- n*2 ) - n cells in bytes. 3 CELLS  ->  6
[x] CHAR+ ( addr -- addr+1 ) - advance one char.
[x] CHARS ( n -- n ) - n chars in bytes (1 byte/char here).
[x] ALIGN ( -- ) - align HERE (no-op on this byte-addressed CPU).
[x] ALIGNED ( addr -- addr ) - align an address (no-op here).

--- durexForth ---
[x] DUMP ( addr -- ) - hex + ASCII dump of 8 lines starting at addr.
[x] N ( -- ) - dump the next block (continues the previous DUMP).
