=== TILE - tilemap cells & layer setup ===

Read/write cells of the layer-1 tilemap (the text screen). Each cell is a
screen/tile code plus a colour attribute; the address follows the current
screen mode.

[x]   TILE  ( x y code attr -- )   write a tile cell
[x]   TDATA ( x y -- code )        read a cell's tile/screen code
[x]   TATTR ( x y -- attr )        read a cell's colour attribute (fg | bg<<4)

VERA layer configuration (either layer 0 or 1):
[x]   LAYER-ON   ( layer -- )            enable a display layer
[x]   LAYER-OFF  ( layer -- )            disable a display layer
[x]   MAPBASE    ( layer bank addr -- )  set tile-map base (bank:addr, 512-aligned)
[x]   TILEBASE   ( layer bank addr -- )  set tile-data base (2 KB aligned)
[x]   LAYER-MODE ( layer cfg -- )        write the layer config byte (map size,
                                     T256C, bitmap mode, colour depth)
[x] TMAPSAVE ( c-addr u -- ) - save the layer-1 tilemap (self-sizing).
[x] TMAPLOAD ( c-addr u -- ) - load the layer-1 tilemap back to its VRAM address.
[x] TILESAVE ( c-addr u vaddr len -- ) - save len bytes of a bank-1 tileset.
[x] TILELOAD ( c-addr u vaddr -- ) - load a tileset into bank-1 VRAM at vaddr.
