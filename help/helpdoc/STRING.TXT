=== STRING - Characters and strings ===

S" ." COUNT TYPE /STRING CHAR [CHAR] TIB #TIB are always resident; the rest
is an on-demand module - cart: NEEDS STRING, SD card: INCLUDE STRING.
Strings built at run time (interpreted S\" and C", CHR, RPT) live in PAD and
stay valid until the next such call (max 189 characters).

[x] S" ( "ccc\"" -- addr len ) - a string literal. S" hello" TYPE  ->  hello
[x] "xxx" ( -- addr len ) - string literal, same semantics as S" but no space
    after the quote: "hello world" TYPE. Only kicks in for undefined tokens
    (defined words always win); an unterminated string errors like an
    unknown word. Handler cell: 'QUOTE (see INTERPRETER).
[x] S\" ( "ccc\"" -- addr len ) - string literal with escapes: \n \t \" \\ \e etc.
[x] ." ( "ccc\"" -- ) - print a literal string when the word runs. : HI ." hello"
    ;
[x] C" ( "ccc\"" -- c-addr ) - a counted-string literal.
[x] COUNT ( c-addr -- addr len ) - convert a counted string to address/length.
[x] TYPE ( addr len -- ) - print a string. S" hi" TYPE
[x] COMPARE ( a1 u1 a2 u2 -- n ) - compare two strings; -1/0/1.
[x] /STRING ( addr len n -- addr+n len-n ) - remove n characters from the front.
[x] PLACE ( addr len dst -- ) - store a string as a counted string at dst.
[x] +PLACE ( addr len dst -- ) - append a string to the counted string at dst.
[x] CHAR ( "c" -- n ) - code of the next character (interpreting). CHAR A  ->  65
[x] [CHAR] ( "c" -- ) - compile the code of the next character (in a definition).
    [CHAR] *
[x] LINPUT ( c-addr +n -- +n2 ) - read a line from the keyboard into a buffer.
[x] NHEX ( u -- c-addr u ) - number as hexadecimal digits. 255 NHEX TYPE  ->  FF
[x] NBIN ( u -- c-addr u ) - number as binary digits. 5 NBIN TYPE  ->  101
[x] STR ( n -- c-addr u ) - signed number as a string (current base). -12 STR TYPE
    ->  -12
[x] VAL ( c-addr u -- n ) - string to number (current base). S" 42" VAL .  ->  42
[x] ASC ( c-addr u -- code ) - code of the first character.
[x] CHR ( code -- c-addr 1 ) - one-character string (in PAD). 65 CHR TYPE  ->  A
[x] LEN ( c-addr u -- u ) - string length (returns the count).
[x] LEFT ( c-addr u n -- c-addr n2 ) - first n characters.
[x] RIGHT ( c-addr u n -- c-addr2 n2 ) - last n characters.
[x] MID ( c-addr u start len -- c-addr2 len2 ) - substring; start is 1-based. S"
    HELLO" 2 3 MID TYPE  ->  ELL
[x] RPT ( char n -- c-addr u ) - a character repeated n times (in PAD). [CHAR] * 5
    RPT TYPE  ->  *
[x] SLITERAL ( addr len -- ) - compile a string literal.
[x] TIB ( -- addr ) - the terminal input buffer.
[x] #TIB ( -- addr ) - variable: number of characters in TIB.
[x] GETKEY ( -- char )   block until a key is pressed, return its PETSCII code.
