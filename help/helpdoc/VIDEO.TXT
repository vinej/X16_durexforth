=== VIDEO - X16 video, screen and cursor ===

VERA is the X16's video chip. VPOKE/VPEEK do single random accesses; for bulk
access set the address once with VADDR then stream with V!/V@.

[x] VPOKE ( bank addr value -- ) - write a byte to VRAM (BASIC VPOKE
    bank,addr,value). 0 $1000 65 VPOKE
[x] VPEEK ( bank addr -- value ) - read a byte from VRAM (BASIC VPEEK(bank,addr)).
[x] VADDR ( bank addr -- ) - point the VERA data port at VRAM (auto-increment 1).
    bank is the 17th address bit.
[x] V! ( byte -- ) - store a byte through the data port (address auto-increments).
[ ] synonym VSTORE V!
[x] V@ ( -- byte ) - read a byte through the data port (address auto-increments).
[ ] synonym VREAD V@
[x] V!W ( w -- ) - store a 16-bit word through the data port, low byte first.
[ ] synonym VWSTORE V!
[x] SCREEN ( mode -- ) - set the video mode. 0=80x60, 1=80x30, 2=40x60, 3=40x30,
    128=320x240x256. 0 SCREEN
[x] COLOR ( fg bg -- ) - set text foreground/background (0-15). 1 6 COLOR
[x] BORDER ( color -- ) - set the display border color (0-15).
[x] CLS ( -- ) - clear the text screen.
[x] LOCATE ( row col -- ) - move the text cursor (BASIC LOCATE row,col).
[x] CURSOR ( -- row col ) - read the cursor position (inverse of LOCATE).
[x] POS ( -- col ) - current cursor column.
[x] SCROLLX ( n -- ) - set layer-1 horizontal hardware scroll (0-4095).
[x] SCROLLY ( n -- ) - set layer-1 vertical hardware scroll (0-4095).
[x] TILE ( x y code attr -- ) - set a tile cell (BASIC TILE x,y,code,attr).
[x] TDATA ( x y -- code ) - read a tile's code (BASIC TDATA(x,y)).
[x] TATTR ( x y -- attr ) - read a tile's attribute (BASIC TATTR(x,y)).
